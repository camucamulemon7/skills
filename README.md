# Skills

Reusable agent skills.

Each skill lives in its own directory and includes a required `SKILL.md` file. Optional resources such as `references/`, `scripts/`, and `assets/` should stay inside the relevant skill directory.

## Skills

- `reduce-rtl-routing-congestion`: Transforms synthesizable RTL to reduce routing congestion while preserving functional equivalence.
- `reduce-rtl-power`: Transforms synthesizable RTL to reduce power while preserving functional equivalence.
- `resolve-rtl-timing-bottlenecks`: Transforms synthesizable RTL to resolve timing bottlenecks while preserving logical equivalence.
